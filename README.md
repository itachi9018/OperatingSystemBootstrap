# Operating System Bootstrap

Generate a complete Obsidian knowledge operating system.

---

## Features

- Creates the complete vault structure
- Generates README files
- Creates Templates
- Creates AI Skills
- Creates Maps of Content
- GitHub friendly
- Markdown only
- Safe to run multiple times

---

## Usage

```powershell
.\bootstrap.ps1
```

or

```powershell
.\bootstrap.ps1 -VaultPath "D:\Obsidian\Operating System"
```
