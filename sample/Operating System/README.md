# Operating System

> A long-term knowledge system for engineering, business, design, systems thinking, and product creation.

## Core Philosophy

Capture quickly.
Think deeply.
Extract principles.
Link ideas.
Build better systems.

Information should flow upward:

Inbox
→ Knowledge
→ Principles
→ Ideas
→ Projects
→ Reflection
→ Philosophy

Never duplicate notes.

Store information once.

Link everywhere.

Capture first.
