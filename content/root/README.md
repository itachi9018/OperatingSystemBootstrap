# Operating System

A personal knowledge operating system.

Designed for:

- engineering growth
- startup building
- product creation
- game development
- systems thinking
- strategic learning

---

# Core Idea

Information should compound.

The purpose of this vault is not storage.

The purpose is transformation:

Information

↓

Understanding

↓

Patterns

↓

Principles

↓

Better Decisions

↓

Better Systems

---

# Operating Rules

## Capture

Capture quickly.

Organize later.

## Extract

Do not collect information endlessly.

Extract meaning.

## Connect

Knowledge becomes valuable through relationships.

## Improve

Every project should permanently upgrade your thinking.

---

# The Main Question

Do not ask:

"What should I write down?"

Ask:

"What pattern exists here?"
